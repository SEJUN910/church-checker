# Project: 엔크리스토 (church-checker)

This is a Next.js 15 church management app using Supabase (Auth, DB, Storage).

## Design System
Always follow the design guidelines in [DESIGN.md](./DESIGN.md) when building or modifying UI.

Key rules at a glance:
- **No 1px borders** — use background color shifts to separate sections
- **Surface palette**: `#fcf9f4` (base) → `#f0ede8` (content) → `#ffffff` (cards)
- **Typography**: Noto Serif for headlines, Plus Jakarta Sans for body
- **Corners**: `rounded-2xl` (1.5rem) for CTAs, `rounded-xl` (1rem) for cards
- **Shadows**: `0px 20px 40px rgba(28, 28, 25, 0.06)` for modals — no harsh shadows
- **Primary color**: `#32617d`, **Gold accent**: use sparingly
- **No pure black** — use `#1c1c19` for text
